# Network Security Assessments Portfolio

This repository contains network security assessment reports and findings
conducted across various network environments and systems. All assessments
were conducted using industry standard network security tools and practices.
As this portfolio grows, security testing will be integrated earlier in the
development lifecycle, moving toward the DevSecOps principle of shifting
security left.


## About

These assessments are part of my ongoing network security practice and
professional development. I am actively building my skills through hands-on
experience using publicly available online testing environments and resources
such as TryHackMe, Hack The Box, and similar platforms. Each assessment
documents vulnerabilities found, tools used, remediation steps taken, and
accepted risk justifications for any remaining findings. This portfolio
reflects my growth as a security practitioner and my commitment to continuous
learning in the network security field.


## Tools and Methodology

| Tool | Type | Purpose |
|---|---|---|
| Nmap | Network Scanner | Port scanning and service detection |
| Wireshark | Packet Analyzer | Network traffic analysis |
| Nessus | Vulnerability Scanner | Network vulnerability scanning |
| OWASP ZAP | DAST | Dynamic application security testing |
| Metasploit | Exploitation Framework | Vulnerability validation |
